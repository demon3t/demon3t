## About

I build practical engineering software: domain models, calculation tools, APIs, and UI packages that make complex systems easier to work with.

My main playground is the `Titanic` ecosystem: C#/.NET backend architecture, entity metadata, HTTP APIs, TypeScript clients, React components, npm packages, and automated publishing through GitHub Actions.

> I like turning messy domain rules into calm, typed systems.

## Current Focus

- Building `Titanic.Entity`: entity metadata, API contracts, client packages, and UI primitives.
- Publishing scoped npm packages under [`@titanic-entity`](https://www.npmjs.com/org/titanic-entity).
- Developing engineering calculation tools for power systems and electrical line analysis.
- Keeping codebases simple, typed, testable, and comfortable to extend.
