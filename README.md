## Обо мне

Я разрабатываю практичные инженерные системы: доменные модели, расчетные инструменты, API, клиентские пакеты и UI-модули, которые помогают работать со сложной логикой спокойнее и быстрее.

Мой основной фокус сейчас - экосистема `Titanic`: серверная архитектура на C#/.NET, Entity ORM, метаданные, HTTP API, TypeScript-клиенты, React-компоненты, NuGet/npm-пакеты и автоматическая публикация через GitHub Actions.

> Люблю превращать сложные предметные правила в понятные типизированные системы.

## Сейчас в фокусе

- Развитие `Titanic.Entity`: метаданные, Entity ORM, API-контракты, клиентские пакеты и UI-примитивы.
- Публикация npm-пакетов в [`@titanic-entity`](https://www.npmjs.com/org/titanic-entity).
- Поддержка NuGet-пакетов [`Titanic.Entity`](https://www.nuget.org/packages/Titanic.Entity) и [`Titanic.Db`](https://www.nuget.org/packages/Titanic.Db).
- Инженерные расчеты для электроэнергетики и длинных линий электропередачи.
- Простая архитектура, типизация, проверяемые сборки и удобная автоматизация.
